Customer Segmentation using K-Means ClusteringThis project implements an end-to-end unsupervised machine learning pipeline to segment retail customers using Mall_Customers.csv. By identifying distinct customer personas, this project translates raw behavioral data into actionable marketing strategies.
■ Task objectiveThe primary goal of this project is to partition a retail customer base into distinct, meaningful groups based on demographics and purchasing behavior. Specifically, the project aims to:Discover Hidden Patterns: Uncover natural groupings within customer attributes (Age, Annual Income, and Spending Score).Dimensionality Reduction: Utilize Principal Component Analysis (PCA) to compress the feature space and visualize multi-dimensional clusters in a 2D plane.Formulate Target Strategies: Provide tailored, data-driven marketing recommendations for each customer segment to maximize business ROI and customer retention.
■ Your approachThe machine learning pipeline is structured through the following systematic phases:Exploratory Data Analysis (EDA): Analyzes the distributions of individual features using histograms and examines feature interactions using a Pearson correlation matrix.Data Preprocessing & Scaling: Isolates the behavioral features and applies StandardScaler to normalize the data, ensuring features with larger magnitudes (like income) do not disproportionately dominate the distance metrics.Optimal Cluster Selection: Implements the Elbow Method, running KMeans across a range of $1$ to $10$ clusters, plotting the within-cluster sum of squares (Inertia) to visually pinpoint the optimal number of clusters ($K=4$).Clustering Execution: Trains the final KMeans algorithm with $K=4$ clusters and extracts the true centroids by shifting them back to their original mathematical scale using an inverse transform.Dimensionality Reduction & Visualization: Applies PCA to project the $3$-dimensional standardized space onto $2$ principal components, tracking total explained variance while plotting the clusters alongside their respective multi-dimensional centroids.Segment Profiling: Group-aggregates demographic profiles (including Gender distributions) for each cluster and generates interactive scatter plots mapping customer age against spending score, scaling data points dynamically by annual income.
